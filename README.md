## 📚 Notebooks

### 📓 1. Multiple Linear Regression
- **Dataset:** Toyota Corolla
- **Goal:** Predict car prices based on multiple features
- **Techniques:** MLR, feature selection, model evaluation
- **Libraries:** Pandas, Matplotlib, Scikit-learn

---

### 📓 2. Logistic Regression
- **Dataset:** Diabetes (Pima Indians)
- **Goal:** Predict whether a patient has diabetes
- **Techniques:** Binary classification, confusion matrix, accuracy score
- **Libraries:** Pandas, Scikit-learn

---

### 📓 3. Data Transformations
- **Dataset:** Adult/Census Income
- **Goal:** Clean and prepare data for ML models
- **Techniques:** Null handling, encoding, normalization, feature engineering
- **Libraries:** Pandas, NumPy
